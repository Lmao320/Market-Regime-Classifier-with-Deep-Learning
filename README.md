# Market-Regime-Classifier-with-Deep-Learning

Sistema cuantitativo que identifica regímenes de mercado con LSTM, capturando dinámicas temporales y cambios estructurales en datos multi-activo.

🚀 Descripción

Modelo de Deep Learning aplicado a finanzas cuantitativas que clasifica el mercado en distintos estados usando:

Retornos
Volatilidad
Correlaciones

Genera probabilidades dinámicas de régimen, permitiendo entender cómo evoluciona el mercado en el tiempo.

⚡ Highlights
LSTM para detección de regímenes
Enfoque híbrido (KMeans + Deep Learning)
Modela dependencias temporales no lineales
Dashboard interactivo (Streamlit + Plotly)

🧠 Qué problema resuelve

Los mercados cambian constantemente.

Este sistema permite:

Detectar cambios de régimen
Identificar condiciones de riesgo
Mejorar decisiones en estrategias cuantitativas
